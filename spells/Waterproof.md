# <font style='color:red'>Waterproof</font> 
**School:** abjuration 
_arcanist_ 1, _wizard_ 1, _sorcerer_ 1, _witch_ 1, _magus_ 1, _bard_ 1, _skald_ 1, _bloodrager_ 1, _shaman_ 1, _druid_ 1, _hunter_ 1, _ranger_ 1, _cleric_ 1, _oracle_ 1, _warpriest_ 1, _alchemist_ 1, _investigator_ 1, 
#### Casting
**Casting Time:** standard action
 **Components:** V, S 
 #### Effects
**Targets:** creature touched (1) or object touched of up to 10 lbs./level (1)
**Range:** touch
**Duration:** 24 hours
**Saving Throw:** Will negates (harmless); **Spell Resistance:** yes (harmless); 
 #### Description
This spell renders a touched object waterproof for the duration of the spell. The object must be no larger than 10 pounds per caster level. The spell protects the object from any mundane liquids (such as alcohol or oil) but not from magically created liquids. If this spell is cast on a creature, all of that creature's equipment is rendered waterproof as long as it is carried by the target creature. If the creature drops or discards an item, the item is no longer protected by this spell, even if the targeted creature picks it up later.Mundane combustibles under the effects of this spell still need exposure to air in order to ignite.
