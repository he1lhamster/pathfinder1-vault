# <font style='color:red'>Minor Phantom Object</font> 
**School:** illusion (figment) [mind-affecting] 
_arcanist_ 4, _wizard_ 4, _sorcerer_ 4, _psychic_ 4, _mesmerist_ 4, 
#### Casting
**Casting Time:** 1 minute
 **Components:** V, S 
 #### Effects
**Range:** 0 ft.
**Duration:** 10 minutes/level (D)
**Effect:** illusory, unattended, nonmagical object of nonliving plant matter, up to 1 cu. ft./level
**Saving Throw:** Will negates; **Spell Resistance:** yes; 
 #### Description
This spell functions as the _[[spells/Minor Creation|minor creation]]_ spell, except the object created is a semi-real illusion. Any creature that interacts with the object may make a Will save, with success causing the object to cease to exist. A gnome casting this spell may make a Spellcraft check in place of any Craft check required to make a complex item.

 #### Other
**Source:** Advanced Race Guide
**Rating:** 1
