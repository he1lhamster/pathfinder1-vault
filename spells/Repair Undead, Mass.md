# <font style='color:orange'>Repair Undead, Mass</font> 
**School:** necromancy 
_arcanist_ 5, _wizard_ 5, _sorcerer_ 5, _spiritualist_ 5, 
#### Casting
**Casting Time:** 1 standard action
 **Components:** V, S 
 #### Effects
**Targets:** one undead creature/level, no two of which can be more than 30 ft. apart
**Range:** close (25 ft. + 5 ft./2 levels)
**Duration:** instantaneous
**Saving Throw:** Will half (harmless); **Spell Resistance:** yes (harmless); 
 #### Description
This spell functions as _[[spells/Repair Undead|repair undead]]_, except as noted above, and it cures 1d8 points of damage + 1 point per caster level (maximum +20) on each selected creature.

 #### Other
**Source:** Advanced Class Guide
