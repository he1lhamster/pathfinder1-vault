_arcanist_ 4, _wizard_ 4, _sorcerer_ 4, _witch_ 4, _bard_ 4, _skald_ 4, _psychic_ 4, _spiritualist_ 3, _medium_ 4, 
#### Casting
**Casting Time:** 1 minute
 **Components:** V, S, M (diamonds worth price of arms to be created) {diamonds worth price of arms to be created}
 #### Effects
**Range:** 0 ft.
**Duration:** instantaneous
**Effect:** one nonmagical weapon, shield, or set of armor
**Saving Throw:** none; **Spell Resistance:** no; 
 #### Description
You create one nonmagical weapon, _[[spells/Shield|shield]]_, or set of armor. To determine whether the item functions as intended, you must attempt an appropriate Craft check with the DC required to create the item normally. If you fail, the item is created with the _[[conditions/Broken|broken]]_ condition. You can create items made of rare or unusual materials if you include a bit of that material as an additional material component when you cast the spell, but cannot create cold iron items with this spell.
