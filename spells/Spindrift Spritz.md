# <font style='color:green'>Spindrift Spritz</font> 
**School:** conjuration (healing) 
_shaman_ 2, _druid_ 2, _hunter_ 1, _ranger_ 1, _cleric_ 2, _oracle_ 2, _warpriest_ 2, _inquisitor_ 2, _paladin_ 1, _spiritualist_ 2, 
#### Casting
**Casting Time:** immediate action
 **Components:** V, S 
 #### Effects
**Targets:** creature (1)
**Range:** close (25 feet + 5 feet/2 levels)
**Duration:** instantaneous
**Saving Throw:** Will negates (harmless); **Spell Resistance:** yes (harmless); 
 #### Description
You can cast this spell only as a response to a target attempting a saving throw against an effect that would cause the _[[conditions/Fatigued|fatigued]]_, _[[conditions/Shaken|shaken]]_, _[[conditions/Sickened|sickened]]_, or _[[conditions/Staggered|staggered]]_ conditions. The target can roll its saving throw twice and take the higher result.
