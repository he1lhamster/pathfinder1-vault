 **Components:** V, S

**Saving Throw:** Will disbelief (if interacted with); **Spell Resistance:** yes; 
 #### Description
This spell functions like _[[spells/Shadow Evocation|shadow evocation]]_, except that it enables you to create partially real, illusory versions of _[[classes/Sorcerer|sorcerer]]_ or _[[classes/Wizard|wizard]]_ evocation spells of 7th level or lower. If recognized as a greater _shadow evocation_, a damaging spell deals only three-fifths (60%) damage.
