# <font style='color:red'>Weird</font> 
**School:** illusion (phantasm) [fear, mind-affecting, emotion] 
_arcanist_ 9, _wizard_ 9, _sorcerer_ 9, _psychic_ 9, 
#### Casting
**Casting Time:** 1 standard action
 **Components:** V, S 
 #### Effects
**Targets:** any number of creatures, no two of which can be more than 30 ft. apart
**Range:** medium (100 ft. + 10 ft./level)
**Duration:** instantaneous
**Saving Throw:** Will disbelief, then Fortitude partial; see text; **Spell Resistance:** yes; 
 #### Description
This spell functions like _[[spells/Phantasmal Killer|phantasmal killer]]_, except it can affect more than one creature. Only the affected creatures see the _[[items/Armor Magic Abilities/Phantasmal|phantasmal]]_ creatures attacking them, though you see the attackers as shadowy shapes.
  If a subject's Fortitude save succeeds, it still takes 3d6 points of damage and is _[[conditions/Stunned|stunned]]_ for 1 round. The subject also takes 1d4 points of Strength damage.

 #### Other
**Source:** PFRPG Core
