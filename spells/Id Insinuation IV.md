# <font style='color:blue'>Id Insinuation IV</font> 
**School:** enchantment (compulsion) [mind-affecting] 
_psychic_ 5, 
#### Casting
**Casting Time:** 1 standard action
 **Components:** S 
 #### Effects
**Targets:** four creatures
**Range:** close (25 ft. + 5 ft./2 levels)
**Duration:** concentration + 1 round
**Saving Throw:** Will negates; **Spell Resistance:** yes; 
 #### Description
This functions as _[[spells/Id Insinuation I|id insinuation I]]_, except as noted above. In addition, on the first round of the effect, you can select the result of each creature's _[[conditions/Confused|confused]]_ condition without having to roll, but this doesn't allow you to make any decisions about how that result is applied. On the following rounds of the effect, the targets of this spell must roll twice to determine their _confused_ effect. You select which roll is used. This spell can be undercast.

 #### Other
**Source:** Occult Adventures
**Rating:** 4
