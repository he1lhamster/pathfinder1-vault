# <font style='color:green'>Eagle Eye</font> 
**School:** divination 
_shaman_ 2, _druid_ 2, _hunter_ 2, _ranger_ 2, _psychic_ 2, 
#### Casting
**Casting Time:** 1 minute
 **Components:** V, S, DF 
 #### Effects
**Range:** long (400 ft. + 40 ft./level)
**Duration:** 1 minute/level (D)
**Effect:** magical sensor
**Saving Throw:** none; **Spell Resistance:** no; 
 #### Description
_[[spells/Eagle Eye|Eagle eye]]_ creates a magical sensor directly above you. The sensor can appear anywhere above you, to a maximum height equal to the spell's range. You can see from this vantage as if you were actually there, rotating your viewpoint 360 degrees.
  You perceive with your normal visual senses. A caster using _eagle eye_ can easily see for a considerable distance. The spell does not penetrate any solid surface, although it is unaffected by foliage and the like.
