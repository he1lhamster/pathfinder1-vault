# <font style='color:orange'>Shield Companion (AA)</font> 
**School:** abjuration 
_arcanist_ 1, _wizard_ 1, _sorcerer_ 1, _witch_ 1, _druid_ 1, _hunter_ 1, _ranger_ 1, _antipaladin_ 1, _paladin_ 1, 
#### Casting
**Casting Time:** standard action
 **Components:** V, S, F 50 / 50
 #### Effects
**Targets:** animal companion, familiar, or fiendish servant (1)
**Range:** close (25 feet + 5 feet/2 levels)
**Duration:** 1 hour/level (D)
**Saving Throw:** Will negates (harmless); **Spell Resistance:** yes (harmless); 
 #### Description
This spell functions as _[[spells/Shield Other|shield other]]_ but affects only the caster's animal companion or familiar. Spellcasters from classes that do not normally gain an animal companion, familiar, or fiendish servant but who gain one through an alternate class feature, archetype, or prestige class can prepare and cast this spell as a 1st-level spell if they are capable of casting spells.
