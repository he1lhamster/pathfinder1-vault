# <font style='color:green'>Constricting Coils</font> 
**School:** enchantment (compulsion) [mind-affecting] 
_arcanist_ 5, _wizard_ 5, _sorcerer_ 5, _cleric_ 5, _oracle_ 5, _warpriest_ 5, 
#### Casting
**Casting Time:** 1 standard action
 **Components:** V, S, M/DF (a snakeskin) {a snakeskin}
 #### Effects
**Targets:** one living creature
**Range:** medium (100 ft. + 10 ft./level)
**Duration:** 1 round/level; see text (D)
**Saving Throw:** Will negates; see text; **Spell Resistance:** yes; 
 #### Description
This spell functions like _[[spells/Hold Monster|hold monster]]_, except that the target is constricted as if by the coils of a large snake, taking 1d6+6 points of bludgeoning damage each round that it fails its save. A successful save ends both the paralyzing and constriction effects.

 #### Other
**Source:** AP 42
**Rating:** 3
