# <font style='color:red'>False Resurrection, Greater</font> 
**School:** conjuration (calling) [evil, lawful, ruse] 
_cleric_ 9, _oracle_ 9, 
#### Casting
**Casting Time:** 1 minute
 **Components:** V, S, M (diamond worth 25,000 gp), DF {diamond worth 25,000 gp}
 #### Effects
**Targets:** dead creature touched
**Range:** touch
**Duration:** permanent
**Saving Throw:** none; **Spell Resistance:** yes (harmless); 
 #### Description
This spell functions as _[[spells/False Resurrection|false resurrection]]_ except that i t calls a belier devil with the young simple template (Pathfinder RPG Bestiary 2 85, 292) to possess the body and all parts of _false resurrection_ that function like _[[spells/Resurrection|resurrection]]_ instead function like _[[spells/True Resurrection|true resurrection]]_. Attempts to _[[spells/Identify|identify]]_ greater _false resurrection_ with a skill check incorrectly _identify_ it as _true resurrection_ (see the ruse descriptor on page 192). A fooled viewer mistakes greater _false resurrection_'s aura as the lingering aura of an instantaneous conjuration effect.

 #### Other
**Source:** Ultimate Intrigue
**Rating:** 1
