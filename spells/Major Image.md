# <font style='color:blue'>Major Image</font> 
**School:** illusion (figment) 
_arcanist_ 3, _wizard_ 3, _sorcerer_ 3, _magus_ 3, _bard_ 3, _skald_ 3, _psychic_ 3, _mesmerist_ 3, _occultist_ 3, _medium_ 3, 
#### Casting
**Casting Time:** 1 standard action
 **Components:** V, S, F (a bit of fleece) {a bit of fleece}
 #### Effects
**Range:** long (400 ft. + 40 ft./level)
**Duration:** Concentration + 3 rounds
**Effect:** visual figment that cannot extend beyond four 10-ft. cubes + one 10-ft. cube/level (S)
**Saving Throw:** Will disbelief (if interacted with); **Spell Resistance:** no; 
 #### Description
This spell functions like _[[spells/Silent Image|silent image]]_, except that sound, smell, and thermal illusions are included in the spell effect. While concentrating, you can move the image within the range.
  The image disappears when struck by an opponent unless you cause the illusion to react appropriately.
