# <font style='color:orange'>Continual Flame</font> 
**School:** evocation [light] 
_arcanist_ 2, _wizard_ 2, _sorcerer_ 2, _cleric_ 3, _oracle_ 3, _warpriest_ 3, _inquisitor_ 3, _psychic_ 3, _occultist_ 2, 
#### Casting
**Casting Time:** 1 standard action
 **Components:** V, S, M (ruby dust worth 50 gp) {ruby dust worth 50 gp}
 #### Effects
**Targets:** object touched Effect magical, heatless flame
**Range:** touch
**Duration:** permanent
**Saving Throw:** none; **Spell Resistance:** no; 
 #### Description
A flame, equivalent in brightness to a _[[items/Mundane/Torch|torch]]_, springs forth from an object that you touch. The effect looks like a regular flame, but it creates no _[[universal monster rules/Heat|heat]]_ and doesn't use oxygen. A _[[spells/Continual Flame|continual flame]]_ can be covered and hidden but not smothered or quenched.
  Light spells counter and dispel _[[spells/Darkness|darkness]]_ spells of an equal or lower level.
