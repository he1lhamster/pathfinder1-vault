# <font style='color:green'>Twilight Haze</font> 
**School:** illusion (shadow) [darkness, shadow] 
_arcanist_ 2, _wizard_ 2, _sorcerer_ 2, _witch_ 2, _psychic_ 2, _mesmerist_ 2, _spiritualist_ 2, 
#### Casting
**Casting Time:** 1 standard action
 **Components:** V, S 
 #### Effects
**Range:** close (25 ft. + 5 ft./2 levels)
**Duration:** 1 round/level
**Effect:** fog spreads in 20-ft. radius
**Saving Throw:** none; **Spell Resistance:** no; 
 #### Description
You create a _[[monsters/Gray|gray]]_ haze that obscures sight (as _[[spells/Fog Cloud|fog cloud]]_) and causes the illumination level within it to dim by one step (as per _[[spells/Darkness|darkness]]_). Since this spell produces an illusion rather than a true physical fog, it is unaffected by wind and can be cast _[[items/Weapon Magic Abilities/Underwater|underwater]]_.

 #### Other
**Source:** Advanced Class Guide
**Rating:** 3
**Potion:** Yes
