# <font style='color:orange'>Steal Size</font> 
**School:** transmutation 
_arcanist_ 2, _wizard_ 2, _sorcerer_ 2, _witch_ 2, _magus_ 3, _bloodrager_ 2, _druid_ 3, _hunter_ 3, 
#### Casting
**Casting Time:** 1 standard action
 **Components:** V, S, M (a piece of iron shaped like a scale tray) {a piece of iron shaped like a scale tray}
 #### Effects
**Targets:** one humanoid creature larger than you
**Range:** close (25 ft. + 5 ft./2 levels)
**Duration:** 1 minute/level (D)
**Saving Throw:** Fortitude negates; **Spell Resistance:** yes; 
 #### Description
This spell reduces the target's size by one size category, as if it were affected by _[[spells/Reduce Person|reduce person]]_. You grow in size by one category, as if affected by _[[spells/Enlarge Person|enlarge person]]_ ( if y ou c an b e targeted by that spell). If the spell is dispelled or otherwise ends for either you or the target, both you and the target return to your normal sizes.

 #### Other

**Potion:** Yes
