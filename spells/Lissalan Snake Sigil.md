# <font style='color:red'>Lissalan Snake Sigil</font> 
**School:** see text 
_arcanist_ 3, _wizard_ 3, _sorcerer_ 3, _cleric_ 3, _oracle_ 3, _warpriest_ 3, 
#### Casting
**Casting Time:** 10 minutes
 **Components:** V, S, M (powdered amber worth 500 gp and a snake scale) {powdered amber worth 500 gp and a snake scale}
 #### Effects
**Targets:** one touched book or written work
**Range:** touch
**Duration:** permanent until discharged; 1 day/level; see text
**Saving Throw:** Reflex negates; **Spell Resistance:** no; 
 #### Description
There are seven variants of this spell, one for each of the Thassilonian schools of magic. Each functions like _[[spells/Sepia Snake Sigil|sepia snake sigil]]_ (and counts as that spell for the purpose of combining other spells that hide or garble text), except instead of trapping the subject, the triggered sigil's effect depends on this spell's school. This effect lasts for 1 day/level. This is a curse effect that can be removed via _[[spells/Remove Curse|remove curse]]_. Abjuration: All beneficial magical effects on the target last half as long as normal. Conjuration: The target is _[[conditions/Nauseated|nauseated]]_. This is a poison effect. Enchantment: The target takes a 1d6 penalty to Intelligence, Wisdom, and Charisma. This is a compulsion effect. Evocation: The target gains _[[curses/Vulnerability|vulnerability]]_ to an energy type, chosen randomly from the following: acid, cold, electricity, or fire. This is an acid, cold, electricity, or fire effect. Illusion: The target's _[[spells/Vision|vision]]_ is blurred, giving it a -4 penalty on Perception checks relating to _vision_, and the target treats all other creatures as having _[[spells/Displacement|displacement]]_. This is a glamer effect. Necromancy: The target is _[[conditions/Exhausted|exhausted]]_. This condition cannot be removed with rest. Transmutation: Target is affected by _[[spells/Slow|slow]]_.
