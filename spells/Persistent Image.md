# <font style='color:blue'>Persistent Image</font> 
**School:** illusion (figment) 
_arcanist_ 5, _wizard_ 5, _sorcerer_ 5, _bard_ 5, _skald_ 5, _psychic_ 5, _mesmerist_ 5, _occultist_ 5, _medium_ 4, 
#### Casting
**Casting Time:** 1 standard action
 **Components:** V, S, F (a bit of fleece) {a bit of fleece}
 #### Effects
**Range:** long (400 ft. + 40 ft./level)
**Duration:** 1 min./level (D)
**Effect:** visual figment that cannot extend beyond four 10-ft. cubes + one 10-ft. cube/level (S)
**Saving Throw:** Will disbelief (if interacted with); **Spell Resistance:** no; 
 #### Description
This spell functions like _[[spells/Silent Image|silent image]]_, except that the figment includes visual, auditory, olfactory, and thermal components, and the figment follows a script determined by you. The figment follows that script without your having to concentrate on it. The illusion can include intelligible speech if you _[[spells/Wish|wish]]_.
