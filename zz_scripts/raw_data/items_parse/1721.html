Description

This blue-black +1 defiant chainmail has a metal badge depicting Calistria's three-dagger symbol fused to it. Each of these suits of armor is crafted with a particular type of foe in mind for the defiant property, as appropriate for the vengeance the original crafter sought. Once per day, the wearer can use charm person (lasting 1 hour).
