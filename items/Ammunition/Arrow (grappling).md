# Arrow (grappling)

### Properties

**Type:** Ammunition **Price:** 1 gp **Weight:** 0.5 lbs. **Slot:** Slotless

### Description

This small _[[items/Weapon/Grappling hook|grappling hook]]_ is designed to be tied to a _[[items/Mundane/Silk rope|silk rope]]_ and fired from a bow. When fired, it has a range increment of 30 feet. An equivalent item exists for crossbows.
