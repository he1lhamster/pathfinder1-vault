# Smoke arrow

### Properties

**Type:** Ammunition **Price:** 10 gp **Weight:** – **Slot:** Slotless

### Description

This arrow is actually a specially-shaped _[[items/Mundane/Smokestick|smokestick]]_ that can be fired from a bow. It trails smoke as it flies, and creates a 5-foot cube of smoke where it strikes. It otherwise functions like a normal arrow in terms of damage, range, and so on.
