# Dragoncrafting kit

### Properties

**Type:** Mundane **Price:** 40 gp **Weight:** 4 lbs. **Slot:** Slotless

### Description

This kit contains all the butchers' tools, preservatives, and desiccants necessary to turn a dragon corpse into usable items. It provides the user with a +2 circumstance bonus on all _[[spells/Heal|Heal]]_ and Craft skill checks made to harvest dragoncraft materials and create dragoncraft items (see page 18).
