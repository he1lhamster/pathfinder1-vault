# Lantern honey

### Properties

**Type:** Mundane **Price:** 150 gp **Weight:** – **Slot:** Slotless

### Description

This alchemically enhanced honey is typically transported in small, lantern-shaped containers capable of holding up to 3 doses of honey. A single dose of _[[items/Mundane/Lantern honey|lantern honey]]_ functions as normal _[[items/Mundane/Antitoxin|antitoxin]]_, except that it also provides _[[universal monster rules/Immunity|immunity]]_ to _[[poisons/Drow poison|drow poison]]_ for 1 hour after _[[feats/Consumption|consumption]]_.
