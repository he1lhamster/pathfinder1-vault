# Surgeon's tools

### Properties

**Type:** Mundane **Price:** 20 gp **Weight:** 5 lbs. **Slot:** Slotless

### Description

When used in conjunction with a _[[items/Mundane/Healer's kit|healer's kit]]_, _[[items/Mundane/Surgeon's tools|surgeon's tools]]_ raise the kit's bonus to a +3 circumstance bonus on _[[spells/Heal|Heal]]_ checks to treat wounds or _[[items/Weapon Magic Abilities/Deadly|deadly]]_ wounds.
