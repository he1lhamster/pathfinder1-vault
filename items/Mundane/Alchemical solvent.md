# Alchemical solvent

### Properties

**Type:** Mundane **Price:** 20 gp **Weight:** 0.5 lbs. **Slot:** Slotless

### Description

This bubbling purple gel eats through adhesives. Each _[[items/Mundane/Vial|vial]]_ can cover a single 5-foot square. It destroys most normal adhesives (such as tar, tree _[[items/Weapon/Sap|sap]]_, or glue) in a single round but takes 1d4+1 rounds to deal with more powerful adhesives (_[[items/Mundane/Alchemical glue|alchemical glue]]_, tanglefoot bags, spider webbing, and so on). It has no affect on fully magical adhesives, such as _[[items/Wondrous Item/Sovereign Glue|sovereign glue]]_.
