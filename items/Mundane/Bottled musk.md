# Bottled musk

### Properties

**Type:** Mundane **Price:** 25 gp **Weight:** 0.5 lbs. **Slot:** Slotless

### Description

This glass _[[items/Mundane/Vial|vial]]_ contains alchemically preserved musk from a single variety of creature. Sniffing _[[items/Mundane/Bottled musk|bottled musk]]_ provides a +4 circumstance bonus on Survival checks to track creatures of the same variety by _[[universal monster rules/Scent|scent]]_. Additionally, _bottled musk_ can be used as an additional spell component for the following spells to modify the spell's effects.

_[[spells/Detect Undead|Detect undead]]_: Functions as written, but detects creatures of the musk's associated creature type.

_[[spells/Locate Creature|Locate creature]]_: Detects the same variety of creature as the musk (for example, using bison musk detects only bison, not all herd animals or animals), and provides a +2 bonus on caster level checks to overcome _[[spells/Nondetection|nondetection]]_.
