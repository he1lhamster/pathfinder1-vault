# Wrist sheath (spring loaded)

### Properties

**Type:** Mundane **Price:** 5 gp **Weight:** 1 lb. **Slot:** Slotless

### Description

This item works like a standard _[[items/Mundane/Wrist sheath|wrist sheath]]_, but releasing an item from it is a swift action. Preparing the sheath for this use requires cranking the sheath's tiny gears and springs into place (a full-round action that provokes an attack of opportunity).
