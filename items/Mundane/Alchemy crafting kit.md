# Alchemy crafting kit

### Properties

**Type:** Mundane **Price:** 25 gp **Weight:** 5 lbs. **Slot:** Slotless

### Description

An _[[classes/Alchemist|alchemist]]_ with an _[[items/Mundane/Alchemy crafting kit|alchemy crafting kit]]_ is assumed to have all the material components needed for his extracts, mutagens, and bombs, except for those components that have a specific cost. An _alchemy crafting kit_ provides no bonuses on Craft (alchemy) checks. (This item was _[[items/Weapon Magic Abilities/Called|called]]_ an "_alchemist_'s kit" in the Advanced Player's Guide, and was renamed to avoid _[[spells/Confusion|confusion]]_ with this book's pre-selected set of adventuring gear _called_ an "_alchemist_'s kit.")
