# Formula book

### Properties

**Type:** Mundane **Price:** 15 gp **Weight:** 3 lbs. **Slot:** Slotless

### Description

A _[[items/Mundane/Formula book|formula book]]_ has 100 pages of parchment, and each alchemical extract formula takes up one page per extract level. A newly purchased _formula book_ contains no extract formulas. An _[[classes/Alchemist|alchemist]]_ character begins play with a _formula book_ containing the extract formulae he knows (Advanced Player's Guide 32). This is the _alchemist_ equivalent of a _[[items/Mundane/Spellbook|spellbook]]_ for wizards (a _spellbook_ and a _formula book_ are essentially the same item).
