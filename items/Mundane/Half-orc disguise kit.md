# Half-orc disguise kit

### Properties

**Type:** Mundane **Price:** 75 gp **Weight:** 1 lb. **Slot:** Slotless

### Description

This specifically designed kit was put together by half-orcs to help them appear more human. It includes skin cream to disguise green-gray skin color and other items designed to conceal or draw attention away from typical half-orc physical characteristics. Using this kit on a half-orc grants a +3 bonus on Disguise checks to appear human. Used on an _[[monsters/Orc|orc]]_, it only grants a +1 bonus on these checks. A _[[items/Mundane/Half-orc disguise kit|half-orc disguise kit]]_ is _[[conditions/Exhausted|exhausted]]_ after 10 uses.
