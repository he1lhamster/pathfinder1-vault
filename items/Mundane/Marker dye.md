# Marker dye

### Properties

**Type:** Mundane **Price:** 15 gp **Weight:** 1 lb. **Slot:** Slotless

### Description

This dye (available in several colors) creates an obvious stain wherever it is splashed. Washing has no effect for the first 72 hours (though magic can _[[spells/Erase|erase]]_ it), but it fades completely after 2 weeks. You can throw it as a splash weapon.
