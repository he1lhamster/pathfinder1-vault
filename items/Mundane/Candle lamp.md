# Candle lamp

### Properties

**Type:** Mundane **Price:** 5 gp **Weight:** 1 lb. **Slot:** Slotless

### Description

A _[[items/Mundane/Candle lamp|candle lamp]]_ is a small lanternlike device with glass panels and a slot to hold a _[[items/Mundane/Candle|candle]]_. It protects the _candle_ from drafts and catches wax drippings. It provides the same light as a _candle_, but can be shuttered such that it only illuminates 1 to 3 adjacent squares. You can use a _candle lamp_ to keep your hands warm.
