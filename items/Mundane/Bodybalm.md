# Bodybalm

### Properties

**Type:** Mundane **Price:** 25 gp **Weight:** – **Slot:** Slotless

### Description

When this pungent yellow _[[items/Mundane/Powder|powder]]_ is boiled in water and given to a creature to drink, it provides the attending _[[npcs/Healer|healer]]_ a +5 alchemical bonus on _[[spells/Heal|Heal]]_ checks for providing long-term care, treating poison, and treating disease.
