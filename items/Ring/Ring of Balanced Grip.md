# Ring of Balanced Grip

### Properties

**Type:** Ring **Price:** 1600 gp **Weight:** – **Slot:** Ring

##### Magical properties

**Caster level:** 7 **Aura:** moderate transmutation

### Description

This sturdy ring allows the user to ignore the nonproficiency penalty for using a particular type of improvised weapon, such as a _[[items/Mundane/Crowbar|crowbar]]_ or pitchfork, and grants a +1 enhancement bonus on attack rolls with that item as though it were a masterwork weapon.

### Crafting requirements

**Crafting cost:** 800 gp

_[[feats/Forge Ring|Forge Ring]]_, _[[spells/Masterwork Transformation|masterwork transformation]]_
