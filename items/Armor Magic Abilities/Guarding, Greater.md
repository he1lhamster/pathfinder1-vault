#  Guarding, Greater

**Source** _Advanced Race Guide pg. 148_
**Aura** strong abjuration **CL** 12th
**Slot** shield quality; **Price** +2 bonus; **Weight** —

### Description

As the _guarding _property, except as a free action, the wearer selects any number of adjacent allies to receive the shield's bonus. All selected allies receive the same bonus. If a shielded target is ever more than 5 feet from the wearer, the effect ends for that target but not for any other targets.

### Construction

**Requirements** Craft Magic Arms and Armor, _shield other_; **Price** +2 bonus
