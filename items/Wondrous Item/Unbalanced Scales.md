# Unbalanced Scales

### Properties

**Type:** Wondrous Item **Price:** 1000 gp **Weight:** 1 lb. **Slot:** Slotless

##### Magical properties

**Caster level:** 3 **Aura:** faint illusion

### Description

A set of _[[items/Wondrous Item/Unbalanced Scales|unbalanced scales]]_ permits an unscrupulous merchant to conduct transactions with inaccurate weight results, but still pass inspections by commerce officials when necessary. The _unbalanced scales_ normally provide an accurate measurement of weight, but by means of a _[[spells/Command|command]]_ word, the user can cause the set of scales to provide a reading that is off (either lighter or heavier, as desired) by as much as 25%.

### Crafting requirements

**Crafting cost:** 500 gp

_[[feats/Craft Wondrous Item|Craft Wondrous Item]]_, _[[spells/Prestidigitation|prestidigitation]]_
