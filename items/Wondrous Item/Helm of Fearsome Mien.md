# Helm of Fearsome Mien

### Properties

**Type:** Wondrous Item **Price:** 5000 gp **Weight:** 4 lbs. **Slot:** Head

##### Magical properties

**Caster level:** 1 **Aura:** faint necromancy

### Description

This metal helm is made to look like the wearer has large fangs and other features resembling those of a fearsome predator. If the wearer is a _[[classes/Barbarian|barbarian]]_, she can use the intimidating glare _[[spells/Rage|rage]]_ power whenever she rages. The helm has no effect if the wearer is not a _barbarian_ (or another class with the _rage_ class feature).

### Crafting requirements

**Crafting cost:** 2500 gp

_[[feats/Craft Wondrous Item|Craft Wondrous Item]]_, _[[spells/Cause Fear|cause fear]]_
