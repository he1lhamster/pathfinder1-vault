# Aspis Badge of Last Resort

### Properties

**Type:** Wondrous Item **Price:** 750 gp **Weight:** – **Slot:** Slotless

##### Magical properties

**Caster level:** 3 **Aura:** faint transmutation

### Description

This badge depicts the Aspis Consortium's insignia: a coiled serpent. By speaking a _[[spells/Command|command]]_ word, the wearer can transform the badge into one of five tools, selected from the following:

* A masterwork _[[items/Shield/Light steel shield|light steel shield]]_ bearing the Aspis insignia
* A masterwork _[[items/Weapon/Whip|whip]]_
* A _[[items/Weapon/Dagger|dagger]]_ poisoned with a dose of _[[poisons/Blue Whinnis|blue whinnis]]_
* A 50-foot _[[items/Mundane/Silk rope|silk rope]]_ with a _[[items/Weapon/Grappling hook|grappling hook]]_
* A set of masterwork thieves' tools
