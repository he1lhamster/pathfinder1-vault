# Sniper Goggles (Greater)

### Properties

**Type:** Wondrous Item **Price:** 50000 gp **Weight:** – **Slot:** Eyes

##### Magical properties

**Caster level:** 8 **Aura:** moderate _[[spells/Divination|divination]]_

### Description

These goggles feature stubby periscopelike lenses that expand and retract depending on the needs of the wearer. These goggles function as _[[items/Wondrous Item/Sniper Goggles|sniper goggles]]_, but the wearer gains the +2 circumstance bonus on each sneak attack when making a sneak attack at any range.
