# Discriminating Cube

### Properties

**Type:** Wondrous Item **Price:** 40000 gp **Weight:** 0.5 lbs. **Slot:** Slotless

### Description

This _[[items/Wondrous Item/Cube of Force|cube of force]]_ only creates one type of screen, which excludes living matter. Each of the five active faces sets the screen to exclude a different type of living matter (using the same categories as the _[[spells/Bane|bane]]_ weapon special ability).
