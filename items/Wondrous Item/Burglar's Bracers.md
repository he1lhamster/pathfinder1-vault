# Burglar's Bracers

### Properties

**Type:** Wondrous Item **Price:** 1050 gp **Weight:** 1 lb. **Slot:** Wrists

##### Magical properties

**Caster level:** 3 **Aura:** faint transmutation

### Description

On _[[spells/Command|command]]_, these plain leather bracers can transform into a set of masterwork thieves' tools. They can transform back to their bracer form with another such _command_. Once per day, while using these thieves' tools or wearing these bracers, the user can take 10 on a single Disable Device check, even when in immediate danger or distracted.

### Crafting requirements

**Crafting cost:** 550 gp

_[[feats/Craft Wondrous Item|Craft Wondrous Item]]_, _[[spells/Shrink Item|shrink item]]_
