# Eyes of Eyebite

### Properties

**Type:** Wondrous Item **Price:** 30000 gp **Weight:** – **Slot:** Eyes

##### Magical properties

**Caster level:** 11 **Aura:** moderate necromancy

### Description

These glasses look like golden teeth holding a pair of greentinted lenses. The wearer may use _[[spells/Eyebite|eyebite]]_ 11 times per day on _[[spells/Command|command]]_, affecting one target for each use of the spell. The wearer does not need to use these rounds consecutively, but each activation is a standard action (unlike the spell, in which targeting a new foe is a swift action). If the wearer removes the glasses, the _eyebite_ effects immediately end.

### Crafting requirements

**Crafting cost:** 15000 gp

_[[feats/Craft Wondrous Item|Craft Wondrous Item]]_, _eyebite_
