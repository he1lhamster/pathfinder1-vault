# Unguent of Timelessness

### Properties

**Type:** Wondrous Item **Price:** 150 gp **Weight:** – **Slot:** Slotless

##### Magical properties

**Caster level:** 3 **Aura:** faint transmutation

### Description

When applied to any matter that was once alive, such as wood, paper, or a dead body, this ointment allows that substance to resist the passage of time. Each year of actual time affects the substance as if only a day had passed. The coated object gains a +1 _[[universal monster rules/Resistance|resistance]]_ bonus on all saving throws. The unguent never wears off, although it can be magically removed (by _[[items/Weapon Magic Abilities/Dispelling|dispelling]]_ the effect, for instance). One _[[items/Mundane/Flask|flask]]_ contains enough material to coat eight _[[classes/Medium|Medium]]_ or smaller objects. A Large object counts as two _Medium_ objects, and a Huge object counts as four _Medium_ objects.

### Crafting requirements

**Crafting cost:** 75 gp
