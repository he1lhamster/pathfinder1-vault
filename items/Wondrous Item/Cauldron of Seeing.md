# Cauldron of Seeing

### Properties

**Type:** Wondrous Item **Price:** 42000 gp **Weight:** 5 lbs. **Slot:** Slotless

##### Magical properties

**Caster level:** 10 **Aura:** moderate _[[spells/Divination|divination]]_

### Description

When filled with liquid, this small _[[items/Mundane/Cauldron|cauldron]]_ allows the user to see over virtually any distance as if using the _[[spells/Scrying|scrying]]_ spell. It may have additional powers like those of a crystal ball for the same prices.
