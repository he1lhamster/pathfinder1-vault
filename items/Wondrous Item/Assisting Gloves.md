# Assisting Gloves

### Properties

**Type:** Wondrous Item **Price:** 180 gp **Weight:** 1 lb. **Slot:** Hands

##### Magical properties

**Caster level:** 1 **Aura:** faint transmutation

### Description

The wearer of these simple gloves can speak a _[[spells/Command|command]]_ word create a glowing disembodied hand. As a swift action, the wearer can have the hand assist him with a task as if using the aid another action. The hand uses the wearer's base attack bonus or skill ranks when making the check to see if aid another is successful (it does not gain the benefit of the wearer's ability scores or other bonuses). The hand remains until it has attempted one action or until 1 minute passes, whichever comes first, at which time the gloves become nonmagical.

### Crafting requirements

**Crafting cost:** 90 gp

_[[feats/Craft Wondrous Item|Craft Wondrous Item]]_, _[[spells/Mage Hand|mage hand]]_, _[[spells/Unseen Servant|unseen servant]]_
