# Hooked axe

### Properties

**Type:** Weapon **Price:** 20 gp **Weight:** 7 lbs. **Slot:** Slotless

##### Weapon properties

**Proficiency group:** Exotic **Type:** One-Handed **Damage:** 1d8 **Critical:** x3 **Damage type:** S **Weapon groups:** Axes **Special:** Disarm, Performance, _[[universal monster rules/Trip|Trip]]_

### Description

The hooked end of this axe's blade can be used to disarm foes or _[[universal monster rules/Pull|pull]]_ them off balance. A _[[items/Weapon/Hooked axe|hooked axe]]_ can be used as a martial weapon (in which case it functions like a _[[items/Weapon/Battleaxe|battleaxe]]_).
