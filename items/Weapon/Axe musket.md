# Axe musket

### Properties

**Type:** Weapon **Price:** 1600 gp **Weight:** 6 lbs. **Slot:** Slotless

##### Weapon properties

**Proficiency group:** Exotic **Type:** Two-Handed **Damage:** 1d8 **Critical:** x4 **Range:** 30 ft. **Damage type:** B and P

### Description

This short _[[items/Weapon/Musket|musket]]_ features an axe blade at the end of its _[[items/Mundane/Barrel|barrel]]_. It can be used as both a _musket_ and a _[[items/Weapon/Battleaxe|battleaxe]]_. It is considered a double weapon for the purposes of creating masterwork or magical versions of this weapon. If this firearm gains the _[[conditions/Broken|broken]]_ condition, both the firearm component and the axe are considered _broken_. An _[[items/Weapon/Axe musket|axe musket]]_ uses either a bullet and a single dose of black _[[items/Mundane/Powder|powder]]_ or an alchemical cartridge as ammunition. This is an early firearm.
