# Great terbutje

### Properties

**Type:** Weapon **Price:** 12 gp **Weight:** 4 lbs. **Slot:** Slotless

##### Weapon properties

**Proficiency group:** Exotic **Type:** One-Handed **Damage:** 1d10 **Critical:** 19-20/x2 **Damage type:** S **Weapon groups:** Heavy Blades **Special:** Fragile

### Description

This Stone Age weapon is a 4-foot-long version of the _[[items/Weapon/Terbutje|terbutje]]_. It is too large to use in one hand without special _[[items/Weapon Magic Abilities/Training|training]]_; thus, it is an exotic weapon. You can use a _[[items/Weapon/Great terbutje|great terbutje]]_ twohanded as a martial weapon.
