# Double-barreled pistol

### Properties

**Type:** Weapon **Price:** 1750 gp **Weight:** 5 lbs. **Slot:** Slotless

##### Weapon properties

**Proficiency group:** Exotic **Type:** One-Handed **Damage:** 1d8 **Critical:** x4 **Range:** 20 ft. **Damage type:** B and P **Weapon groups:** Firearms

### Description

This _[[items/Weapon/Pistol|pistol]]_ has two parallel barrels; each _[[items/Mundane/Barrel|barrel]]_ can be fired independently as a separate action, or both can be shot at once with the same action. If both barrels are shot at once, they must both target the same creature or object, and the _pistol_ becomes wildly inaccurate, imposing a –4 penalty on each shot. This is an early firearm.
