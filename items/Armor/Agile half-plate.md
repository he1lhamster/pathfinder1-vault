# Agile half-plate

### Properties

**Type:** Armor **Price:** 850 gp **Weight:** 55 lbs. **Slot:** Armor

##### Armor properties

**Type:** Heavy **Armor bonus:** 8 **Maximum dexterity:** 0 **Armor check penalty:** -7 **Arcane spell failure chance:** 40

### Description

This style of _[[items/Armor/Half-plate|half-plate]]_ is specially crafted in a manner that allows extra maneuverability for some physical activities. The armor check penalty for _[[universal monster rules/Climb|Climb]]_ checks and _[[spells/Jump|jump]]_ checks is only –4 (masterwork and mithral versions of this armor reduce this penalty as well as the normal penalty). In addition, unlike with most heavy armors, the wearer can still run at quadruple speed instead of triple speed.
