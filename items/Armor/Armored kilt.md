# Armored kilt

### Properties

**Type:** Armor **Price:** 20 gp **Weight:** 10 lbs. **Slot:** Armor

##### Armor properties

**Type:** Light **Armor bonus:** 1 **Maximum dexterity:** 6 **Armor check penalty:** 0 **Arcane spell failure chance:** 0

### Description

When you add an _[[items/Armor/Armored kilt|armored kilt]]_ to a suit of light armor, the set counts as _[[classes/Medium|medium]]_ armor. Likewise, a _[[items/Mundane/Kilt|kilt]]_ and _medium_ armor counts as heavy armor. Adding an _armored kilt_ to heavy armor has no effect.
