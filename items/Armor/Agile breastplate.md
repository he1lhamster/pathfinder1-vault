# Agile breastplate

### Properties

**Type:** Armor **Price:** 400 gp **Weight:** 25 lbs. **Slot:** Armor

##### Armor properties

**Type:** _[[classes/Medium|Medium]]_ **Armor bonus:** 6 **Maximum dexterity:** 3 **Armor check penalty:** -4 **Arcane spell failure chance:** 25

### Description

This _[[items/Armor/Breastplate|breastplate]]_ is specially crafted in a manner that allows extra maneuverability for some physical activities. The armor check penalty for _[[universal monster rules/Climb|Climb]]_ checks and Acrobatics checks made to _[[spells/Jump|jump]]_ is only –1 (masterwork and mithral versions of this armor reduce this penalty as well as the normal penalty).
